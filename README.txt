# Zombie Dice Game

- Jogo produzido para a disciplina de Raciocínio Computacional 1 no curso de Análise e Desenvolvimento de Sistemas da PUCPR, no ano de 2022. 

- O jogo deve ser desenvolvido ao longo de 9 semanas, respeitando os requisitos semanais de desenvolvimento.
